# Geo Prospector 6.7.19.2

**Release date:** 20 July 2026

**Edition:** Established Edition

**Platform:** Windows 64-bit

Geo Prospector 6.7.19.2 adds centralized membership and credit management.

## Membership management

- Registers membership automatically when a license is generated.
- Synchronizes customer, plan, billing, balance, expiration, and account status.
- Supports administrative credit additions.
- Supports remote membership blocking.
- Restores valid membership state after application restart.

## Security

- Ed25519-signed license token.
- Device fingerprint verification.
- Embedded public verification key.
- Clock rollback protection.
- Signed security-anchor manifest.
- Protected Flutter, Node.js, Playwright, and Python components.
- Private signing seed excluded from the release.
